# AnyList

1. Clonar proyecto   

2. Ejecutar:
```bash
pnpm install
```

3. Clonar el archivo `.env.template` y renombrarlo a `.env` con
```bash
cp .env.template .env
```
4. Cambiar las variables de entorno

5. Levantar la base de datos
```bash
docker-compose up -d
```

6. Ejecutar modo dev
```bash
pnpm run start:dev
```

7. Ejecutar seed en GraphQL `http://localhost:9999/graphql`
